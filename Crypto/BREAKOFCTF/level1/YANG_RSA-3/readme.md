# RSA

## 題目
$p, q, p+2, q+2$ 都是質數
$n_1 = pq$
$n_2 = (p+2)(q+2)$


## 解法
我們想求的是
$\varphi(n_1) = (p-1)(q-1) = pq - (p+q) + 1$
$\varphi(n_2) = (p+1)(q+1) = pq + (p+q) + 1$


我們目前知道的是
$n_1 = pq$
$n_2 = (p+2)(q+2) = pq + 2(p+q) + 4$

$p + q = \frac{n_2 - n_1 - 4}{2}$

得到 $\varphi(n_1), \varphi(n_2)$ 後就可以找出私鑰，算出密文了
