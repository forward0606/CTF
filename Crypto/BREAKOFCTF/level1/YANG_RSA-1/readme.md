# RSA

## 演算法
### 公鑰與私鑰
1. 找兩個質數 p and q
2. 算出 r = (p - 1) * (q - 1)
3. 找一個與 r 互質的數 e，並且根據 e 找出 d 使的 $ed \equiv 1\ (\mod m)\ $
4. 公鑰為 (N, e)，私鑰為 (N, d)
### 加密
0. 將鑰加密的訊息轉換為數字 m
1. 做 $c = m^e \mod n$，並傳送 c
### 解密
1. 原始的 m 會等於 $c^d \mod n$
2. 將 m 轉換回文字
## 破解
use  http://factordb.com/ 將 n 分解得到 3 個質數，算出歐拉函數
有歐拉函數之後就可以找到 d ，並且用 n 算出答案
